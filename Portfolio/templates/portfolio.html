  • House Price Prediction

    This project predicts house prices based on the location, number of bedrooms, bathrooms, and square footage. It is designed to help real estate agents and homeowners get an accurate estimate of a property's value. The frontend was created using HTML, CSS, and Bootstrap, and the backend was developed using Python.

  • Recommendation System

    This project was designed to help customers make informed purchase decisions. It shows the best purchases made by a particular customer based on their purchase history. The user inputs the date and customer ID, and the system generates recommendations. HTML, CSS, and Bootstrap were used for the frontend, while the backend was developed using Python.

  • Computer Vision Text Recognizer

    This project is a computer vision text recognizer that converts handwritten text to digital text. It uses morphology for better accuracy. The frontend was created using HTML, CSS, and Bootstrap, while the backend was developed using Python.

    View Project

  • Computer Vision Item Counter

    This project is a computer vision item counter that counts the number of items in given freezer images. It detects and locates all the compartments of the refrigerator using computer vision techniques and then counts the items. The user inputs the path to the image, and the program counts and displays the information about the number of items in the image. The frontend was created using HTML, CSS, and Bootstrap, while the backend was developed using Python.
